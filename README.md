## Quick Start (AI Auto News)

See the full guide at [`ai-auto-news/docs/getting-started.md`](ai-auto-news/docs/getting-started.md).

```bash
cd ai-auto-news
npm install
cp .env.example .env.local
npm run dev
```

Open `http://localhost:3000`.
